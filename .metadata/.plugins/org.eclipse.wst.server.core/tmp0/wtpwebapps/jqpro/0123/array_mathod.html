		slice(start, end) : start ~ end - 1 까지의 요소를 잘라서 새로운 배열을 만든다.
		
		splice() : 배열요소를 삭제 변경
		var fruits = ["Banana", "Orange", "Apple", "Mango"];
		fruits.splice(2, 0, "Lemon", "Kiwi");
		---> 2번째 위치에서 0개를 지우고(지우지 않고) "Lemon", "Kiwi"를 삽입한다. 		
		fruits.splice(2, 1, "Lemon", "Kiwi");
		---> 2번째 위치에서 1개를 지우고 "Lemon", "Kiwi"를 삽입한다. 		
		
	




	배열합치기/배열문자열로 변환하기

	a.join() : a배열과 b배열이 하나의 문자열로 변환.
