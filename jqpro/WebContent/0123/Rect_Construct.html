생성자 함수를 이용한 객체생성과 사용

function Rect(width, height) {
		this.width = width;
		this.height = height;

		this.getArea = function() {
			return this.width * this.height;
		}
}

r1 = new Rect();
	객체의 속성 width, height의 값이 정해지지 않으면
	undefined가 발생.
	r1객체의 속성에 값을 하나씩 대입한다.
	r1.width = 10;
	r1.height = 4;
		
r2 = new Rect(5, 7);

객체의 속성에 정해지지 않았어도 동적으로 추가 해서 사용할 수 있다.
